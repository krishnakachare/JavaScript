    /* What are Cookies ?    
    -Cookies let you store user information in web pages (document pages).
    -Cookies are data, stored in small text files.
    -When a web server has sent a web page to a browser, the connection is shut down, and the      server forgets everything about the user.
    -Cookies were invented to solve the problem "how to remember information about the user":   
      1. When a user visits a web page, his / her name can be stored in a cookie.
      2. Next time the user visits the page, the cookie "remembers" his / her name. */

  JavaScript can create, read, and delete cookies with the document.cookie property.  
    document.cookie = "username=Krishna";
